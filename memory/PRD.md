# BK Decomart — Premium Home Décor Website

## Original Problem Statement
Build the ultimate premium website for **BK Decomart** (Madurai, since 1995) — a luxury home-décor brand
with the original logo preserved. Feel like Apple / Airbnb / Bentley / Architectural Digest.
Tagline: *Come, Let's Dressup Your Home.*
Contact: +91 97877 13802 · bkdecomartdigital@gmail.com · 12 Sivagangai Main Road, Gomathipuram, Madurai – 625020.

## User Choices (from ask_human)
- **AI Room Visualizer**: Gemini Nano Banana (`gemini-3.1-flash-image-preview`) via Emergent LLM key.
- **Photography**: Pre-generate luxury interior images with Nano Banana.
- **Admin Dashboard**: Skipped for now.
- **Enquiries**: WhatsApp deep-link only (no backend storage).
- **Scope**: Complete cohesive MVP end-to-end.

## Architecture
- **Frontend**: React 19 + Tailwind + shadcn/ui + framer-motion (via CSS).
- **Fonts**: Bodoni Moda (serif) + Manrope (sans) — distinctive luxury pairing.
- **Colour tokens**: Ivory `#FAF9F6`, Beige `#F5F0E6`, Champagne `#D4AF37`, Walnut `#4B3621`, Matte `#1A1A1A`.
- **Backend**: FastAPI single endpoint `/api/visualize` — accepts image + product + style → Nano Banana edit.
- **Static images**: served from `/api/static/img/generated/*.png` (FastAPI StaticFiles).
